#### 注释

python 的单行注释使用#开头

```py
# 这是第一个hello程序
print("hello")
```

python 的多行注释用三个引号包裹，可以是单引号也可以是双引号

```py

"""
这里是多行注释
使用双引号来注释
"""
'''
这里是多行注释
使用单引号
'''
print("hello world!!!")

```
