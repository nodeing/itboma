### 常用的快捷键

快捷键是帮助我们提升开发效率的，下面这些快捷键你可以先看一遍，能记住就记住，记不住在后续编码的过程中再慢慢记

```
ctrl + d   复制当前行到下一行
ctrl + y   删除当前行，你也可以使用 ctrl + x，剪贴，万一删除错还可以粘贴回来
ctrl + alt + l 格式化代码
ctrl + /   单行注释
ctrl + shift + /  多行注释
ctrl + z   撤销
alt+shift+上箭头   把当前行的代码往上移动
alt+shift+下箭头   把当前行到代码往下移动
```
